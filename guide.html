NETAX Desk란

자주 쓰는 업무 사이트를 폴더별로 모아둔 북마크 런처입니다. 매번 즐겨찾기를 뒤지는 대신, 필요한 폴더만 열어 바로 이동할 수 있도록 만들었습니다.

폴더 펼치기 · 접기

  • 마우스를 올리면 해당 폴더가 임시로 펼쳐집니다. 마우스를 벗어나면 자동으로 다시 접힙니다.
  • 폴더 제목을 클릭하면 그 폴더가 고정되어, 마우스를 다른 곳에 두어도 계속 펼쳐진 상태로 유지됩니다.
  • 고정은 한 번에 하나의 폴더만 가능합니다. 다른 폴더를 클릭하면 기존 고정은 자동으로 풀리고 새 폴더가 고정됩니다.
  • 고정된 폴더를 다시 클릭하면 고정이 해제됩니다.

검색

  • 상단 검색창에 키워드를 입력하면(예: 홈택스, 양도, Claude) 폴더 이름 또는 링크 제목·주소에 해당 단어가 포함된 항목만 남고 나머지는 자동으로 숨겨집니다.
  • 검색어와 일치하는 폴더는 자동으로 펼쳐진 상태로 표시됩니다.
  • 일치하는 항목이 하나도 없으면 "검색 결과가 없습니다" 안내가 표시됩니다.
  • 검색창을 비우면 원래 상태(고정된 폴더만 펼쳐진 상태)로 돌아갑니다.

폴더 구성

현재 등록된 폴더는 다음과 같습니다.

  • 🤖 AI · 🌐 Google · 💾 Storage
  • 👥 고객관리 · 🏪 매장관리
  • ⚖️ 법령공식 · 📚 법령정보
  • 🏢 부동산 · 🗺️ 지도
  • 🧪 실험-AI세무사 · 💻 실험-코딩
  • 💼 업무 · 🧰 업무도구
  • 📡 정보수집 · 📝 참고블로그 · 🔎 포털검색

팁

  • 모든 링크는 새 탭에서 열립니다. NETAX Desk 탭은 그대로 두고 계속 재사용하면 편리합니다.
  • 브라우저 시작 페이지로 지정해두면, 브라우저를 열 때마다 업무 런처가 바로 뜨도록 만들 수 있습니다.
  • 자주 쓰는 폴더는 클릭해서 고정해두면, 매번 마우스를 올릴 필요 없이 항상 펼쳐진 상태로 사용할 수 있습니다.
